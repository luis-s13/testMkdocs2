Install Python:

	python-3.7.4.exe

Development environment:

	pip install mkdocs==0.17.5
	pip install pymdown-extensions==4.11
	pip install markdown==2.6.11
	pip install pygments==2.2.0
	pip install mkdocs-material==2.9.2
	pip install mkdocs-awesome-pages-plugin==1.2.0

Test:

	mkdocs serve
	
		(exposes the web site on localhost:8000)

	mkdocs build

		(builds the static files (html, css, and js) of the web site)

Build MD articles from Word:

	Writage-1.12.msi
